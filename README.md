# 🎓 Student Performance Predictor

A Machine Learning web application that predicts a student's math score based on demographic and academic factors.

## 🚀 Features

- Student score prediction
- Data preprocessing pipeline
- Machine Learning model training
- Flask web application
- AWS Elastic Beanstalk deployment
- CI/CD using AWS CodePipeline

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Scikit-Learn
- Flask
- AWS Elastic Beanstalk
- AWS CodePipeline

MIproject/

## ⚙️ Installation

```bash
git clone [https://github.com/Dakshhhhh-ops/MIproject.git](https://github.com/Dakshhhhh-ops/Mlproject)
cd MIproject

pip install -r requirements.txt
python application.py
```

## 🌐 Deployment

The application is deployed on AWS Elastic Beanstalk with automated CI/CD using AWS CodePipeline.
